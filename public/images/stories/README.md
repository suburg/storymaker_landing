# Обложки историй

## Размещение изображений

Поместите обложки историй в эту папку со следующими именами:

- `story-1.jpg` (или .png, .webp) - Приключения капитана Макса
- `story-2.jpg` - Тайна пропавших звёзд  
- `story-3.jpg` - Волшебный лес и фея Лили

## Рекомендации

**Формат:** JPG, PNG или WebP  
**Размер:** Оптимально 600x800px (соотношение 3:4)  
**Вес:** До 200KB для оптимальной загрузки

## Как это работает

Компонент `StoryCard.vue` автоматически проверяет наличие изображения:
- Если файл существует → показывает реальную обложку
- Если файла нет → показывает градиентный placeholder с эмодзи

## Пример использования

Чтобы активировать отображение реальных обложек:

1. Добавьте изображения в эту папку
2. В `GallerySection.vue` измените функцию `getStoryImage()`:
   ```typescript
   const getStoryImage = (storyId: string) => {
     return `/images/stories/${storyId}.jpg`
   }
   ```
3. Перезапустите dev сервер

Изображения будут доступны по пути: `/images/stories/story-X.jpg`
